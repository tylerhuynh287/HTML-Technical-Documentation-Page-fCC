Introduction

A brief history of HTML:

  • The World Wide Web has only been in existence for about 30 years. A relatively short time span for what we have today.
  • The current version we use today is the 5th version of HTML otherwise called HTML5, which is what will be shown briefly in this page.
  • If you are curious on learning more about the history of the world wide web, you can read it at this link.

    HTML is the basic language used to code most websites consumers use on a daily basis. It's the language that's used to deliver the basic interface that shows on a page. HTML has a low barrier of entry and doesn't require any extensive prior knowledge to learn some of the most basics of code to create a simple page.
